# ht

### Requirement
- Postgresql
- Node > 6.0.0
- NPM version 3.8.6

### How to start
- clone the project `git clone..`
- run npm install `npm i`
- start the server `npm start`

### Plugins
- Joi
- Vision

### Routes
- [GET] /documentation  : Hapi Swagger
- [GET] /docs           : Routes documentation
